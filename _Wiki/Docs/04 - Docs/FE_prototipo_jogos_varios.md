# Jogos: Cobrinha, Candy, Paciência, Damas/Xadrez (resumo)

## Objetivo
Documentar variantes rápidas de layout e regras visuais.

## Cobrinha (Snake)
- Board responsivo com aspect-ratio:1/1.
- Pontuação destacada em cartão lateral.

## Candy (simplificado)
- Grid 8x8 com gap; itens redondos e grandes.

## Paciência / Spider
- Área de cartas com pilhas; cartas dimensionadas para leitura.

## Damas / Xadrez
- Board 8x8; células quadradas (usar aspect-ratio), peças grandes legíveis.

## Notas gerais
- Usar transform com cuidado; respeitar prefers-reduced-motion.
- Manter controles grandes e acessíveis.
