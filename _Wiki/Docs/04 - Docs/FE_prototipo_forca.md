# Jogo: Forca

## Objetivo
Fornecer interface limpa para adivinhar letras com dica e contador de tentativas.

## Componentes
- Palavra com underscores
- Teclado virtual
- Indicador de tentativas
- Dica

## CSS recomendado
```css
.hangman-word { font-size: 2rem; /* 32px */ }
.hangman-key {
  padding: 0.5rem 0.75rem; /* 8px 12px */
  border-radius: 0.5rem; /* 8px */
}
```

## Notas
- Usar estados desabilitados visuais para teclas já usadas.
