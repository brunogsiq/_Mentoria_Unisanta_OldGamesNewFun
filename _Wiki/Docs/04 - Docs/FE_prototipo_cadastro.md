# Tela: Cadastro

## Objetivo
Fluxo de criação de conta com verificação mínima e feedback em linha.

## Componentes
- Formulário (nome, e-mail, senha)
- Checkbox de termos
- Botão enviar

## CSS recomendado
```css
.signup-card { padding: 3rem; /* 48px */ max-width: 40rem; /* 640px */ }
.signup-input { width: 100%; padding: 0.75rem; /* 12px */ }
```

## Notas
- Validar e-mail no client-side antes de enviar.
