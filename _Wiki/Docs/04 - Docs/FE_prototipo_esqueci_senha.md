# Tela: Esqueci a Senha

## Objetivo
Permitir recuperação de conta via e-mail com fluxo claro.

## Componentes
- Campo para e-mail
- Botão Enviar
- Mensagem de confirmação

## CSS recomendado
```css
.forgot-password-form { max-width: 40rem; /* 640px */ padding: 2rem; /* 32px */ }
.forgot-password-input { width: 100%; padding: 0.75rem; /* 12px */ }
```
